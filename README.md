# WSL2-Linux-Kernel

## Introduction

自用，编译最新的 WSL2 内核。

-   [Linux Kernel](https://www.kernel.org)
-   [WSL2-Linux-Kernel](https://github.com/microsoft/WSL2-Linux-Kernel)

1. 修改内核配置，支持 nbd 模块
